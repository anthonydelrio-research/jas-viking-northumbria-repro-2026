# Repository Scope and License Notes

## Scope

This repository is a reproducibility code package for JASC26-162.

It includes:
- analysis code,
- derived result tables used for manuscript metrics,
- reproducibility documentation.

It does not include redistribution of restricted third-party raw datasets.

## Third-party data constraints

Data providers and terms may include PAS, KEPN/EPNS, CASSS, and OS-derived layers.
Users must obtain and use source data in line with provider licensing terms.

## Reproducibility model

- **Tier 1 (included):** verify manuscript headline metrics from packaged derived tables.
- **Tier 2 (licensed inputs):** rerun complete pipeline using the documented input layer contract.
